# API Pokedesk

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Configuration

[Populate DB](http://localhost:3000/seed)

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov

docker-compose up -d

http://localhost:3000/api/seed
```

**Clonar el archivo .env.example**
```
cp .env.example .env
```
